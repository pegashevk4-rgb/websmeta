# ВебСмета - калькулятор стоимости разработки сайта

Мини-инструмент для менеджера digital-агентства: позволяет быстро оценить примерный бюджет и сроки создания сайта по заданным параметрам.

## Что умеет калькулятор

Калькулятор учитывает:

- тип сайта: лендинг, корпоративный сайт, интернет-магазин;
- количество страниц;
- дополнительные функции: форма заявки, каталог, онлайн-оплата, интеграция с CRM, мультиязычность;
- срочность проекта.

На основе выбранных параметров показывает:

- примерную стоимость проекта;
- ориентировочный срок выполнения;
- краткое пояснение, из чего складывается оценка.

## Структура проекта

- `index.html` — главная страница «агентства» с описанием услуги и ссылкой на калькулятор.
- `calculator.html` — страница калькулятора.
- `styles.css` — общие стили для калькулятора.
- `script.js` — логика расчёта стоимости и сроков.

## Как запустить локально

1. Склонировать репозиторий:
   ```bash
   git clone https://github.com/pegashevk4-rgb/websmeta.git
   cd websmeta
   ```
2. Открыть `index.html` или `calculator.html` в браузере (достаточно дважды кликнуть по файлу или открыть через локальный сервер).
